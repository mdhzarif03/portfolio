## About

This is the personal portfolio of **Muhammad Hasan Zarif**, a self-taught student programmer from Bangladesh.  
The portfolio showcases:

- Profile and skills in **Software, AI, Robotics, and Automation**  
- Verified **certifications** including Harvard CS50x and The Forage programs  
- Selected **projects** with GitHub links  

The portfolio is designed with **Tailwind CSS**, uses **PDF.js** for displaying certificates, and is deployed on **GitHub Pages**.
